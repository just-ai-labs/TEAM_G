## Setup and Installation
1. Clone the repository:
    ```bash
    git clone <https://github.com/just-ai-labs/TEAM_G.git>
    ```
2. Navigate to the project directory:
    ```bash
    cd TEAM_G
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Configure environment variables:
    ```bash
    cd .env.local
    ```
5. Run the project:
    ```bash
    npm run dev
    ```
